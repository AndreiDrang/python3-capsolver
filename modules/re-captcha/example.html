ReCaptcha¶

To import this module:

from python3_capsolver.recaptcha import ReCaptcha
class python3_capsolver.recaptcha.ReCaptcha(captcha_type: ReCaptchaV2TypeEnm | ReCaptchaV3TypeEnm | str, websiteURL: str, websiteKey: str, pageAction: str | None = None, *args, **kwargs)¶

The class is used to work with Capsolver ReCaptcha methods.

Parameters:

  • api_key – Capsolver API key

  • captcha_type (ReCaptchaV2TypeEnm | ReCaptchaV3TypeEnm | str) – Captcha type name, like ReCaptchaV2Task and etc.

  • websiteURL (str) – Address of a webpage with Google ReCaptcha

  • websiteKey (str) – Recaptcha website key. <div class=”g-recaptcha” data-sitekey=”THAT_ONE”></div>

  • pageAction (str | None) – Widget action value. Website owner defines what user is doing on the page through this parameter. Default value: verify. Example: grecaptcha.execute(‘site_key’, {action:’login_test’}).
