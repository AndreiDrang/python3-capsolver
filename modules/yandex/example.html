YandexCaptcha¶

To import this module:

from python3_capsolver.yandex import YandexCaptcha
class python3_capsolver.yandex.YandexCaptcha(api_key: str, captcha_type: CaptchaTypeEnm | str = CaptchaTypeEnm.YandexCaptchaTaskProxyLess, **kwargs)¶

The class is used to work with Capsolver YandexCaptchaTaskProxyLess methods.

Parameters:

  • api_key (str) – Capsolver API key

  • captcha_type (CaptchaTypeEnm | str) – Captcha type name, like YandexCaptchaTaskProxyLess and etc.

  • kwargs –

    additional params for client, like captcha waiting time available keys:

    • sleep_time: int - captcha solution waintig time in sec

    • request_url: str - API address for sending requests,

      else official will be used
